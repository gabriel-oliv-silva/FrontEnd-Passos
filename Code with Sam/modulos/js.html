Variáveis são como "caixas" onde guardamos informações para usar depois. No JavaScript moderno, utilizamos let (para valores que podem mudar) e const (para valores constantes/fixos).

Os principais Tipos de Dados são:

  • String: Textos (ex: "Olá Mundo").
  • Number: Números inteiros ou decimais (ex: 25, 3.14).
  • Boolean: Verdadeiro ou Falso (true ou false).
JS let idade = 25; // number (pode mudar)
idade = 26; // válido

const nome = "Sam"; // string (fixo)
nome = "João"; // ERRO! Não se muda uma constante

let estaLogado = true; // boolean

Como fazemos o código tomar decisões ou repetir tarefas? Usando estruturas de controle de fluxo.

  • Condicionais (if/else): Executa um bloco de código SE a condição for verdadeira, SENÃO, executa outro.
  • Loop (for): Repete um bloco de código um número específico de vezes.
  • Loop (while): Repete um bloco ENQUANTO uma condição for verdadeira.
JS // Condicional
let hora = 14;
if (hora < 12) {
console.log("Bom dia!");
} else {
console.log("Boa tarde!");
}
// Loop for (inicialização; condição; incremento)

for (let i=1; i <=3; i++) {
console.log("Contagem: " + i);
}

Funções são blocos de código que realizam uma tarefa específica e podem ser reutilizados (chamados) várias vezes, evitando que você repita código.

Existem duas formas principais de escrever funções modernas:

  • Declaração Tradicional: Usa a palavra function.
  • Arrow Functions: Uma sintaxe mais curta e limpa usando =>. Muito comum em Callbacks (funções passadas como parâmetro para outras).
JS // Função Tradicional
function somar(a, b) {
return a + b;
}

// Arrow Function (faz exatamente a mesma coisa)
const somarArrow = (a, b) => {
return a + b;
};

// Se tiver só uma linha, pode ser ainda menor:
const dobrar = x => x * 2;

Arrays (Vetores) são listas que guardam múltiplos valores em uma única variável. Arrays possuem "poderes" embutidos, chamados de métodos, para manipulá-los facilmente.

  • map(): Cria um novo array transformando todos os itens originais.
  • filter(): Cria um novo array apenas com os itens que passarem num teste.
  • push() / pop(): Adiciona ou remove itens do final da lista.
JS let notas = [5, 8, 10, 4];
// Usando o método filter() com uma Arrow Function
let aprovados = notas.filter(nota => nota >= 7);
console.log(aprovados); // Resultado: [8, 10]

// Usando o método map() para dar ponto extra
let notasExtras = notas.map(nota => nota + 1);

Enquanto arrays são listas, Objetos representam coisas do mundo real agrupando propriedades (chaves) e valores. São definidos usando chaves {}.

O JSON (JavaScript Object Notation) é um formato de texto derivado dos objetos JS, usado para enviar e receber dados entre um site e um servidor.

JS // Criando um Objeto
let usuario = {
nome: "Ana",
idade: 28,
admin: true
};

// Acessando propriedades
console.log(usuario.nome); // "Ana"

// Convertendo o objeto para JSON (Texto) para enviar a um servidor
let textoJSON = JSON.stringify(usuario);

O DOM (Document Object Model) é a representação do seu HTML na memória do navegador. O JavaScript usa o objeto document para "entrar" no HTML, pegar elementos, alterá-los ou criar novos.

  • document.querySelector(): Seleciona um elemento usando seletores CSS (ex: ".minha-classe").
  • document.createElement(): Cria um novo elemento (tag) do zero.
  • elemento.textContent: Altera o texto visível.
JS const titulo = document.querySelector('#domTitulo');

// Quando chamado, muda o texto e o CSS via JS
function mudarTexto() {
titulo.textContent = "Modificado pelo JS! 🎉";
titulo.style.color = "#d4b106";
}

A web é interativa porque o JS fica "escutando" as ações do usuário (cliques, digitação, rolar a página). Esses são os Eventos.

Usamos o método addEventListener() para atrelar uma função a uma ação do usuário.

JS // Criando uma Lista de Tarefas (To-Do)
function adicionarTarefa() {
const input = document.querySelector('#inputTarefa');
const lista = document.querySelector('#minhaLista');

if (input.value === '') return;

const novoItem = document.createElement('li');
novoItem.className = 'list-group-item';
novoItem.textContent = input.value;

lista.appendChild(novoItem); // Adiciona na tela
input.value = ""; // Limpa o campo
}

O JavaScript pode se comunicar com outros computadores (servidores) para buscar dados (como a previsão do tempo, ou a lista de filmes da Netflix). Isso leva tempo. Por isso, usamos a programação Assíncrona.

A função fetch() faz a requisição à API. Como não sabemos quando a resposta chega, ela retorna uma Promise (Promessa). Para esperar por ela, usamos async e await.

JS // Função Assíncrona buscando um conselho aleatório
async function buscarConselho() {
// Await: "Espere a internet responder..."
const resposta = await fetch('https://api.adviceslip.com/advice');
const dados = await resposta.json();

console.log(dados.slip.advice);
}
