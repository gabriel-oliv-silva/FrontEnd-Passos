Variáveis são como "caixas" onde guardamos informações para usar depois. No JavaScript moderno, utilizamos let (para valores que podem mudar) e const (para valores fixos).

Os principais Tipos de Dados são:

  • String: Textos (ex: "Olá Mundo"). Sempre entre aspas.
  • Number: Números (ex: 25, 3.14). Sem aspas.
  • Boolean: Verdadeiro ou Falso (true ou false). 
let idade = 25; // number (pode mudar)
idade = 26; // válido

const nome = "Sam"; // string (fixo)
nome = "João"; // ERRO! Não se muda uma constante

let estaLogado = true; // boolean

Como fazemos o código tomar decisões ou repetir tarefas? Usando estruturas de controle de fluxo.

  • Condicionais (if/else): Executa um bloco de código SE a condição for verdadeira, SENÃO, executa outro.
  • Loop (for): Repete um bloco de código um número específico de vezes.
  • Loop (while): Repete um bloco ENQUANTO uma condição for verdadeira. 
// Condicional
let hora = 14;
if (hora < 12) { 
  console.log("Bom dia!"); 
} else { 
  console.log("Boa tarde!");
}

// Loop for (inicialização; condição; incremento)
for (let i = 1; i <= 3; i++) { 
  console.log("Contagem: " + i);
}
// Loop while (condição)
while (hora > 13) { 
  console.log("É tarde!"); //Printa no console que é tarde, enquanto a hora for maior que 13
}

Funções são blocos de código que realizam uma tarefa específica e podem ser reutilizados (chamados) várias vezes, evitando que você repita código.

Existem duas formas principais de escrever funções modernas:

  • Declaração Tradicional: Usa a palavra function.
  • Arrow Functions: Uma sintaxe mais curta e limpa usando =>. Muito comum no React e em callbacks. 
// Função Tradicional
function somar(a, b) {
  return a + b;
}

// Arrow Function (faz exatamente a mesma coisa)
const somarArrow = (a, b) => {
  return a + b;
};

// Se tiver só uma linha, o "return" é automático:
const dobrar = x => x * 2;

Arrays (Vetores) são listas que guardam múltiplos valores em uma única variável. Arrays possuem "poderes" embutidos, chamados de métodos, para manipulá-los facilmente.

  • map(): Cria um novo array transformando todos os itens originais.
  • filter(): Cria um novo array apenas com os itens que passarem num teste (filtro).
  • push() / pop(): Adiciona ou remove itens do final da lista.
let notas = [5, 8, 10, 4];

// Usando o filter() com uma Arrow Function
let aprovados = notas.filter(nota => nota >= 7);
console.log(aprovados); // Resultado: [8, 10]

// Usando o map() para dar ponto extra
let notasExtras = notas.map(nota => nota + 1);

Enquanto arrays são listas, Objetos representam coisas do mundo real, agrupando características através de propriedades (chaves) e valores. São definidos usando chaves {}.

O JSON (JavaScript Object Notation) é um formato de texto derivado dos objetos JS. É a língua universal usada para enviar e receber dados entre um site e um servidor (API).

// Criando um Objeto
let usuario = {
  nome: "Ana",
  idade: 28,
  admin: true
};

// Acessando propriedades usando o "ponto" (.)
console.log(usuario.nome); // Imprime: "Ana"
console.log(usuario.idade); // Imprime: 28

// Convertendo o objeto para JSON (Texto) para enviar pela rede
let textoJSON = JSON.stringify(usuario);

O DOM (Document Object Model) é a representação do seu HTML na memória do navegador. O JavaScript usa o objeto document para "entrar" no HTML, buscar elementos na tela, alterá-los ou criar novos.

  • document.querySelector(): Seleciona um elemento HTML usando a mesma sintaxe do CSS (ex: ".minha-classe" ou "#meu-id").
  • elemento.textContent: Altera o texto visível dentro de uma tag.
  • elemento.style: Altera o CSS do elemento diretamente via JavaScript.
// 1. Busca o elemento com o ID "meuTitulo" no HTML
const titulo = document.querySelector('#meuTitulo');

// 2. Modifica o texto
titulo.textContent = "Hackeado pelo JS! 🚀";

// 3. Modifica a cor (CSS)
titulo.style.color = "red";

A web é viva porque o JS fica "escutando" as ações do usuário (cliques, digitação, rolar a página). Esses são os chamados Eventos.

Usamos o método addEventListener() para "grudar" uma função a uma ação de um elemento.

// Criando uma Lista de Tarefas (To-Do)
function adicionarTarefa() {
  const input = document.querySelector('#inputTarefa');
  const lista = document.querySelector('#minhaLista');

  // Se o campo estiver vazio, não faz nada
  if (input.value === '') return;

  // Cria um novo item (
  • )
  const novoItem = document.createElement('li');
  novoItem.className = 'list-group-item';
  novoItem.textContent = input.value;

  // Adiciona na tela e limpa o campo
  lista.appendChild(novoItem);
  input.value = "";
}

    O JavaScript pode se comunicar com outros computadores na internet para buscar dados em tempo real (como a previsão do tempo, cotação do dólar ou listas de filmes). O nome desse mensageiro é API.

    Como a internet pode demorar para responder, usamos a programação Assíncrona. O JS diz: "Vou pedir os dados e continuar trabalhando. Quando a resposta chegar, eu aviso!"

    • fetch('url'): É o comando que "vai buscar" os dados na API.
    • await: Significa "espere". Diz ao JS para pausar e esperar a resposta da internet chegar antes de ir para a próxima linha.
    • async: Toda função que usa await precisa ter a palavra async antes dela.
    // 1. A função precisa ser "async"
async function buscarFilmes() {
  
  // 2. Esperamos (await) o fetch buscar os dados
  const resposta = await fetch('https://api.exemplo.com/filmes');
  
  // 3. Esperamos a conversão dos dados para JSON (formato de objeto JS)
  const dados = await resposta.json();

  console.log(dados);
}
